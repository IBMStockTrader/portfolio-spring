# Get Summary Portfolios

Get all portfolios in summary form (without stocks).

**URL** : `/portfolio/`

**METHOD** : `GET`

**Auth Required** : YES

**Permissons Required** : STOCKTRADER, STOCKVIEWER

## Success Response

**Code** : `200 OK`

**Content examples**

```json
[
   {
      "owner" : "fred",
      "total" : 0,
      "sentiment" : "Unknown",
      "nextCommission" : 0,
      "free" : 0,

      "commissions" : 0,
      "balance" : 50
   },
   {
```
